# Buildbox

## Installation

Install the gem

    $ gem install buildbox

Then authenticate

    $ buildbox auth:login [apikey]

Then you can start monitoring for builds like so:

    $ buildbox monitor:start

For more help with the command line interface

    $ buildbox --help
